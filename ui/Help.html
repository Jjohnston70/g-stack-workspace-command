🔄 Label Migration Overview

The Label Migration tool helps you transition from old, unorganized Gmail labels to the new structured label system.

Why Migrate Labels?

  • Consolidate scattered labels into organized categories
  • Move emails from old labels to new hierarchical structure
  • Clean up duplicate or inconsistent labeling
  • Preserve all your emails while reorganizing
Safe Process: Migration moves emails to new labels but doesn't delete old labels. You can verify everything worked before manually deleting old labels.
📋 How to Use Label Migration
Step 1: Scan Labels
Scans all your Gmail labels and populates the migration sheet
Menu: Gmail Tools > Label Migration > Scan Labels
Step 2: Review Suggestions
System auto-suggests new labels based on keywords
Review the "Suggested New Label" column and adjust as needed using the dropdown
Step 3: Select Labels to Migrate
Check the "Move Emails?" box for labels you want to migrate
Only checked labels will be processed during migration
Step 4: Preview & Execute
Preview changes, then execute the migration
Menu: Gmail Tools > Label Migration > Preview Migration, then Execute Migration

▶️ Function Runner Overview

The Function Runner provides a safe, user-friendly way to execute any available function without opening the Apps Script editor.

Why Use Function Runner?

  • Run any function by simply checking a box
  • See all available functions with descriptions
  • Confirmation dialogs for destructive operations
  • Execution logging with timestamps and results
  • No need to access Apps Script editor
Safety First: Functions marked "Requires Confirm" will always show a confirmation dialog before running. Destructive functions are highlighted in red.
🚀 How to Use Function Runner
Step 1: Initialize
Create the Function Runner sheet with all available functions
Menu: Setup & Initialize > Initialize Function Runner
Step 2: Enable Trigger
Set up the checkbox trigger for automatic execution
Menu: Setup & Initialize > Setup Function Runner Trigger
Step 3: Run Functions
Check the "Run" box next to any function to execute it
The function runs immediately, and results appear in the Last Result column
Important: The trigger must be set up once for checkbox execution to work. Without the trigger, you can still run functions from the menu.

📊 Working with the Sheets

Many features are controlled by editing data in the sheets. Here's how:

Auto-Label Rules Sheet

Control which emails get which labels automatically.

Adding a New Rule
  1. Go to the Auto-Label Rules sheet
  2. Add a new row at the bottom
  3. Enter the label name (e.g., "Clients/Buyers")
  4. Add email patterns to match in "From Contains"
  5. Or add subject keywords in "Subject Contains"
  6. Check the "Active" checkbox
Example: To auto-label emails from your lender:
Label: Vendors/Lenders
From Contains: @mylender.com,

Response Templates Sheet

Add your own email templates.

Adding a New Template
  1. Go to the Response Templates sheet
  2. Add a new row
  3. Enter: Template Name, Subject, Body, Use Case
  4. Use {{Name}} for the recipient's name

Captured Leads Sheet

Manage leads captured from email.

Working with Leads
  • Click the Status dropdown to update lead status
  • Click the Email Link to open the original email
  • Add notes in any empty columns you create
  • Sort by Date Captured to see newest first

Label Sorting Prefix

To control where labels appear in Gmail's sidebar:

  1. Open gmail-label-creator.gs
  2. Find LABEL_PREFIX: ''
  3. Change to LABEL_PREFIX: 'zz-' to sort at bottom
  4. Or use LABEL_PREFIX: '00-' to sort at top
  5. Run "Delete All Created Labels" then "Create Gmail Labels"
