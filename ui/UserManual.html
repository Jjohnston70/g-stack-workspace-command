Drive Folder Tools

Create Folder Structure

Creates an organized folder hierarchy in your Google Drive.

Drive Tools > Create Folder Structure

Scan My Drive

Catalogs all files with size, owner, and organization recommendations.

  • Scans up to 1,000 files per run
  • Creates inventory and summary sheets
  • Identifies duplicates and large files

Analysis Tools

ToolWhat It Does
Find DuplicatesLists files with same name and size
Find Large FilesLists files over 10MB

Gmail Label Tools

Create Gmail Labels

Creates color-coded labels customized for {{INDUSTRY_NAME}} businesses.

Gmail Tools > Create Gmail Labels

Auto-Label Rules

Automatically labels incoming emails based on sender and subject.

To customize rules:

  1. Go to the Auto-Label Rules sheet
  2. Edit rules or add new rows
  3. Check the Active box to enable

Label Migration

Migrate from old Gmail labels to the new organized structure.

  1. Scan Labels - find all existing labels
  2. Review Suggestions - check automated matches
  3. Select Labels - check which to migrate
  4. Execute Migration - move the emails
Note: "Delete All Created Labels" only removes labels created by this tool. Your existing labels are safe.
Function Runner

Run any function by clicking a checkbox - no coding required!

Setup (One Time)

  1. Setup & Initialize > Initialize Function Runner
  2. Setup & Initialize > Setup Function Runner Trigger

How to Use

  1. Go to the Function Runner sheet
  2. Find the function you want to run
  3. Check the Run checkbox
  4. Function executes immediately
  5. Result appears in Last Result column

Troubleshooting

Common Issues

ProblemSolution
Menu not showing Refresh the page (Ctrl+R or Cmd+R)
Permission errors Re-authorize when prompted; check Extensions > Apps Script > Triggers
Functions timing out Drive scans limited to 1,000 files; run on smaller batches
Labels not in Gmail Refresh Gmail (F5); run "List Existing Labels" to verify
Auto-label not working Check that trigger is set up; verify rules are Active
